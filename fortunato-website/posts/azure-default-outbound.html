Azure retiring the default outbound access for VMs

Azure will be retiring the default outbound access for VMs on September 30, 2025.

If your virtual machines currently rely on default outbound access, they will continue to function for now. Any VMs created after September 30, 2025, will not have default outbound access. To be honest, that is great because I always liked AWS‚Äôs approach better,every instance has an explicit outbound path. Azure is now aligning to it, which makes networking more predictable, secure, and aligned with best practices.

Recommendation
